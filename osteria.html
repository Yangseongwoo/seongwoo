Background & Goal

기존의 단일 배경의 색상, 정적인 레이아웃 무엇보다 오로지 사이트를 통해서
해당 매장까지 사용자들을 도달하게 해야 하는 점에서 많이 부족하다고 느껴 리디자인을 진행하였습니다.

Strategy & Positioning

다른 경쟁사들은 물론, 음식의 비주얼 그리고 브랜드 컬러의 적절한 활용으로
해당 섹션에서 강조하고 싶은 부분들은 어떤 것인지 정확하게 전달하고 있었습니다.
그리하여 조금 더 브랜드 컬러의 활용과 명확한 인터렉션 기능으로 감성적인 측면에 초점을 두고 진행하였습니다.

As is - To be

전체적인 레이아웃 구성, 배열이나 브랜드 컬러의 사용을 가장 큰 문제점으로 인식하고
사용자에게 정보를 한눈에 전달할 수 있게 리디자인을 진행하였습니다.

CONTENTS INFO

Main page

시작 랜딩 페이지에서 메인 이미지가 계속 바뀌게 해놓았고,
이미지나 사용자가 해당 섹션에서 집중해야 할 요소에 인터렉션, 브랜드 컬러를 적용해서 강조했습니다.

Product page

해당 페이지에서는 애니메이션 기능을 제품 사진에 추가하여,
해당 제품을 사이트에서 보고 매장까지 사용자의 방문을 유도할 수 있도록 디자인하였습니다.

RESULT

최종적으로 레이아웃의 정리, 퍼스널 컬러와 애니메이션 기능을 통한 강조를 통해
사용자가 해당 매장, 매장의 제품에 관심을 더 가질 수 있게 진행하였습니다.
