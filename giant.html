Background & Goal

기존의 정적이고 올드한 느낌으로 구성되어있었던 디자인은 조금 더 사용자들이
제품에 흥미를 느낄 수 있도록 역동적이고 현대적인 느낌을 주기 위해서 리디자인을 진행하였습니다.

Strategy & Positioning

영상, 인터렉션, 사용자와 상호작용하는 기능들을 경재 사이트에서는 많이 사용하고 있었기 때문에
해당 자전거라는 제품의 특성을 잘 나타내기 위한 조금 더 감성적이고 동적인 포지션을 잡고 진행하였습니다.

As is - To be

3가지 정도의 문제점이 있었고 저는 강조되는 브랜드 컬러와, 정보 제공, 시각적 제공에 있어서
동적으로 사용자들에게 제공할 수 있는 방법에 대해 집중했습니다.

CONTENTS INFO

Main page

메인 페이지는 슬라이드 화면, hover와 toggle, 인터렉션 기능을 사용하여
사용자가 사이트와 더 직접적으로 상호작용할 수 있도록 디자인하였습니다

Product page

자전거를 클릭하면 자전거가 움직이면서 해당 정보가 보이는 등,
해당 제품에 사용자가 온전히 관심을 가질 수 있도록 디자인하였습니다.

Result

Giant 자전거의 역동적인 느낌과 해당 브랜드의 파란색이 강조되는 느낌으로 디자인하였습니다.
또한 HOVER와 CLICK EVENT를 활용해 사용자가 해당 사이트에 흥미를 느끼고 탐색할 수 있도록 구성했습니다.
