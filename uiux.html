Date

2024.03.16 ~ 2024.04.16

Overview

개인화에 최적화되어있는 앱 서비스 를 조금 더 사용자가 편리하게 사용할 수 있도록 리디자인을 진행하였습니다.

UI UX Redesign

앱의 서비스, 개인에 맞추다

Overview

맥도날드 앱은 혜택이나 서비스 이용을 위해 앱을 사용하지만 앱에서의 개인화된 서비스가
부족했었고 앱 사용성에 있어서 불편함이 많아 사용자의 편의성이 떨어진다는 부분에 집중했습니다.

User Research

구글 플레이스토어 리뷰 와 실제 해당 어플리케이션을 사용하는 사용자를 대상으로 인터뷰를 진행해 불편함과 앱 사용에 있어서 중요한 점을 알아봤습니다.

Problems & Solution

앞선 조사를 통해 핵심적인 문제점을 도출하고 그에 대한 해결방법을 제시해봤습니다. 주요 문제점은 콘텐츠의 구도 정리와 개인화된 서비스였습니다.

Design Style Guid

개인화와 무엇보다 최적화된 서비스 제공을 위한디자인 스타일 가이드를 작성했습니다.

IA & Wireframe

최대한 불필요한 과정을 없애고 정보 전달이나 사용에 있어서 신속하고 편리하게 사용할 수 있게 와이어프레임을 제작했습니다.

Result

전체적인 목적에 맞게 Main page, Order page, Coupon use page등을 리디자인하였습니다.
